      About

      Narrator with Subtitles is a web application to aid in language learning. It is especially useful for languages that have good textual resources but not enough beginner-friendly resources to practice listening skills. It speaks out text using a text-to-speech engine, and offers control to adjust speech speed and navigate across sentences. It optionally shows translations too.

      To use this web application, you must upload a txt, html, or csv file containing the text you want spoken. Documentation regarding file formats and source code can be found at https://github.com/sharmaeklavya2/narrator-ws.
